# Password Strength Checker
A real-time password strength analyzer — zero dependencies, runs in the browser.

## Features
- 9 security rules checked live
- 4-tier rating: Weak / Moderate / Strong / Very Strong
- Crack time estimate
- Pure HTML + CSS + JS

## Usage
Just open `index.html` in any browser. No install needed.

## Rules Checked
Length · Uppercase · Lowercase · Number · Special char · No repeats · No sequences · Not a common password
